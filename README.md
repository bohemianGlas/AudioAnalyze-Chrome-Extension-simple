動画のようにChromeで再生中の動画・音声のピークレベルと位相チェックをできます

用途
メーターの見方がわかれば画面を見るだけで位相や音声の大きさのエラーを把握できる
（１マンオペレーションの配信でも音声エラーを把握しやすくなる）
リアルタイムで自分の声や配信の音声エラーのチェックは原理的にしにくいが客観的なメーターとして把握できる


問題

ある程度は動くがエラーが出ているので完璧ではない
ピークレベルのスケールなどを調節する必要がある
GUIも改善の余地がある
