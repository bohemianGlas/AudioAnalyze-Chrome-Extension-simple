動画のようにChromeで再生中の動画・音声のピークレベルと位相チェックをできます

用途
メーターの見方がわかれば画面を見るだけで位相や音声の大きさのエラーを把握できる
（１マンオペレーションの配信でも音声エラーを把握しやすくなる）
リアルタイムで自分の声や配信の音声エラーのチェックは原理的にしにくいが客観的なメーターとして把握できる


問題

ある程度は動くがエラーが出ているので完璧ではない
ピークレベルのスケールなどを調節する必要がある
GUIも改善の余地がある

利点

著作権保護された音声以外はChromeで再生可能なものならば配信のプレビュー画面や動画配信サイトの音声、ラジオ放送などの音声エラーのチェック（あれ、この音なんかおかしくないか？と思ったときに自分の環境のせいなのか切り分けができる）またWindowsなどのOSのミキサーを通す前の信号なのでOSのボリューム設定に依存しないピークレベルの監視が可能である

つまり１マンオペレーション、小規模な配信企画、個人視聴のさいに経験でなく客観的なメーターを見ながらエラーをチェックし切り分けができる
